# Werewolf
# 🐺 Loup-Garou : Narrateur Pro
Une application web **légère, moderne et 100% autonome** conçue spécifiquement pour aider le narrateur (MJ) à gérer ses parties de Loup-Garou. Fini les trous de mémoire ou les erreurs de "qui a tué qui" !
## ✨ Fonctionnalités clés
 * **👥 Gestion Persistante :** Ajoutez vos joueurs une seule fois. La liste est sauvegardée automatiquement dans votre navigateur (LocalStorage).
 * **🎭 Distribution Automatique :** Cochez les rôles spéciaux (Loup, Voyante, Sorcière, etc.) et l'app s'occupe de distribuer les rôles aléatoirement aux joueurs.
 * **🌑 Révélation Sécurisée :** Un écran noir plein écran permet de montrer son rôle à un joueur sans que les voisins ne voient quoi que ce soit.
 * **❤️ Gestion du Couple (Cupidon) :** Liez deux joueurs. Si l'un meurt, l'autre est automatiquement éliminé avec une alerte de rappel.
 * **🎖️ Système de Maire :** Désignez le Maire avec un badge visuel unique. En cas de décès, l'app vous rappelle qu'il doit nommer un successeur.
 * **💀 Suivi de Vie :** Grisez les joueurs morts pour garder une vision claire de qui est encore en jeu pendant la nuit.
## 🚀 Comment l'utiliser ?
### 1. Installation
L'application ne nécessite aucune installation.
 * Enregistrez le code dans un fichier nommé index.html.
 * Ouvrez ce fichier avec votre navigateur préféré (Chrome, Safari, Firefox) sur votre **smartphone, tablette ou PC**.
### 2. Préparation (Phase de Configuration)
 1. **Saisissez les noms** des participants et appuyez sur "Ajouter" (ou la touche Entrée).
 2. **Cochez les rôles** que vous souhaitez inclure dans la partie.
 3. Cliquez sur **"Lancer la Partie"**.
### 3. La Nuit 0 (Attribution)
 1. Demandez au village de fermer les yeux.
 2. Appelez chaque joueur un par un.
 3. Cliquez sur l'icône **👁️** à côté de son nom pour lui révéler son rôle en plein écran.
 4. Cliquez sur **"Fermer"** avant de passer au joueur suivant.
### 4. Gestion de la partie
 * **Tuer/Ressusciter :** Cliquez sur le bouton **💀** pour éliminer un joueur.
 * **Cupidon :** Cliquez sur "❤️ Couple", puis sélectionnez les deux amoureux sur le plateau.
 * **Maire :** Cliquez sur "🎖️ Élire Maire", puis sélectionnez le joueur choisi par le village.
## 🛠️ Détails techniques
 * **Langages :** HTML5, CSS3, JavaScript (Vanilla).
 * **Stockage :** localStorage (données conservées même après fermeture de l'onglet).
 * **Confidentialité :** Aucune donnée n'est envoyée sur un serveur. Tout reste local sur votre appareil.
